# Simulación SIR Paralela en Grilla 2D

Este repositorio forma parte del proyecto final de la asignatura de Programación Paralela. Aquí se modela la propagación de una enfermedad sobre una población representada como una grilla 2D (una persona por celda), comparando dos enfoques: una versión secuencial y otra paralela.

El objetivo principal es analizar cómo la paralelización impacta el tiempo de ejecución sin alterar el comportamiento del modelo, además de evaluar el rendimiento mediante experimentos de strong scaling.

Se simula la evolución de la epidemia durante 365 días, incorporando aleatoriedad en las transiciones (enfoque tipo Monte Carlo) para reflejar escenarios más realistas.
